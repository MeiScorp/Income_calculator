# Income_calculator
A Streamlit-based Income Calculator that helps users estimate monthly and yearly net income. The app includes tax calculation, currency selection, income–expense visualization, yearly forecasting, and a built‑in history log for saving results. Clean, interactive, and ideal for quick financial insights.
